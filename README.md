# Address.Book

An example address book project for a work sample.

## Getting Started

### Installation

    git clone https://github.com/AhmetCavus/Address.Book.git
    cd Address.Book
    npm install

### Run
    
    npm start

### Test

    npm test

### Open

    http://localhost:8080/questions
